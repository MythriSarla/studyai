# StudyAI

AI-Powered Personalized Learning Platform.

StudyAI lets students upload study materials (PDF, DOCX, TXT) and automatically generates AI summaries, flashcards, quizzes, weak topic analysis, and personalized study schedules — powered by Groq's Llama 3.3 70B model.

## Tech Stack

- **Frontend:** React, Tailwind CSS, Chart.js
- **Backend:** Python Flask, REST API, JWT Auth
- **Database/Storage:** Firebase Firestore, Firebase Auth, Firebase Storage
- **AI:** Groq API (Llama 3.3 70B Versatile)
